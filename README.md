# amazon-cucumber-testng-automation

## Objective
Automate a workflow on a web application to demonstrate your Selenium expertise in handling dynamic elements.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Maven
- Page Object Model (POM)

## Scenario Automated
1. Navigate to Amazon homepage
2. Search for a product 
3. Click on the first search result
4. Validate product details page using "Add to Cart" button
5. Capture screenshot of product page

## Prerequisites
- Java JDK 8 or higher
- Maven
- Chrome browser
- ChromeDriver configured in system PATH

## How to Execute
1. Clone the repository 
2. Open the project in IntelliJ/Eclipse
3. Run `TestRunner.java` as TestNG test

## Reports
- Cucumber HTML report generated in `target/cucumber-report.html`
